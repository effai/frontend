# frontend

Version: 0.1.0
Test project for Atlys

# Set up 

- `npm install --global yarn`
- `git clone https://github.com/effai/frontend.git`
- `cd frontend`
- `yarn`
- `yarn dev`

## Scripts

- `yarn dev`: Runs the development server (via `next dev`).
